# 贡献指南 / Contributing Guide

感谢您对本项目的兴趣！本项目的初衷是学习交流，欢迎提出建议和改进。

## 项目性质声明

⚠️ **本项目是教育性质的开源项目，仅供学习研究使用，不构成投资建议。**

## 如何贡献

### 报告问题

如果您发现了bug或有改进建议，欢迎通过以下方式反馈：

1. **GitHub Issues**: 提交Issue描述问题
2. **邮件反馈**: 发送邮件至项目维护者

### 提交代码

1. Fork 本仓库
2. 创建您的功能分支 (`git checkout -b feature/AmazingFeature`)
3. 提交您的更改 (`git commit -m 'Add some AmazingFeature'`)
4. 推送到分支 (`git push origin feature/AmazingFeature`)
5. 打开一个 Pull Request

### 代码规范

- 使用 Kotlin 进行开发
- 遵循 [Kotlin Coding Conventions](https://kotlinlang.org/docs/coding-conventions.html)
- 保持代码简洁清晰，添加必要的注释
- 遵循 MVVM 架构模式

### 文档改进

欢迎改进项目文档，包括：
- README 翻译
- 使用说明完善
- 代码注释补充

## 开发环境

### 要求

- Android Studio Hedgehog (2023.1.1) 或更高版本
- JDK 17
- Android SDK 34
- Kotlin 1.9.22

### 构建

```bash
./gradlew clean assembleDebug
```

## 注意事项

1. **不要提交敏感信息**：如 API Keys、密码等
2. **保持学习性质**：本项目旨在学习交流，请勿添加诱导投资的功能
3. **风险提示**：任何新功能都应包含适当的风险提示
