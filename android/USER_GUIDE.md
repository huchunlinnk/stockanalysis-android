# Android 股票分析应用 - 使用指南

## 🎯 快速开始

### 1. 首次启动配置

应用首次启动会显示引导页面。完成引导后，**必须**先配置以下设置才能正常使用：

#### 必需配置:
1. **LLM API 设置** (必须)
   - 点击右上角 ⚙️ 设置图标
   - 选择 LLM 提供商 (OpenAI/Anthropic/Deepseek等)
   - 输入 API Key
   - 输入 Base URL (或使用默认)
   - 输入 Model 名称
   - 点击"保存"

2. **数据源配置** (可选但推荐)
   - 输入 Tushare Token (如有)
   - 点击"测试连接"验证

### 2. 主要功能

## 📱 底部导航栏功能

### 🏠 首页 (Home)
**功能**:
- 股票搜索: 输入股票代码或名称搜索
- 搜索结果: 点击跳转到股票详情页
- 市场概览: 查看主要市场指数

**使用方法**:
1. 在搜索框输入股票代码(如: 600000)或名称(如: 浦发银行)
2. 点击搜索结果跳转到详情页
3. 在详情页可以查看K线图、技术指标、开始分析等

### ⭐ 自选股 (Watchlist)
**功能**:
- 自选股列表管理
- 实时行情查看
- 快速分析入口

**使用方法**:
1. 点击右下角 + 按钮添加自选股
2. 输入股票代码搜索并添加
3. 下拉刷新更新行情
4. 点击股票卡片查看详情或开始分析

### 📊 分析 (Analysis)
**功能**:
- 单股智能分析
- 多Agent协同分析
- 分析模式选择

**使用方法**:
1. 输入股票代码
2. 选择分析模式:
   - 快速分析: 基础面分析
   - 标准分析: 技术+基本面
   - 完整分析: 全面深度分析
   - 多Agent投票: 多模型协同分析
3. 点击"开始分析"
4. 等待分析完成查看结果

### 📋 历史 (History)
**功能**:
- 查看历史分析记录
- 信号准确率对比
- 历史趋势分析

**使用方法**:
1. 浏览历史分析列表
2. 点击记录查看详情
3. 点击"信号对比"查看准确率统计

### 💰 持仓 (Portfolio) - 新增
**功能**:
- 持仓管理
- 交易记录
- 盈亏统计
- 风险分析

**使用方法**:
1. 点击右下角 + 按钮添加交易
2. 选择交易类型(买入/卖出)
3. 输入股票代码、数量、价格
4. 系统自动计算盈亏和风险指标
5. 下拉刷新更新持仓市值

**查看信息**:
- 总资产 = 持仓市值 + 现金
- 盈亏 = 市值 - 成本
- 风险指标: 胜率、盈亏比、最大回撤

## 🔧 设置功能

点击右上角 ⚙️ 图标进入设置页面

### LLM 配置
**支持的提供商**:
- OpenAI (GPT-3.5/GPT-4)
- Anthropic (Claude)
- Deepseek
- 阿里云 (通义千问)
- 智谱AI (GLM)
- Moonshot (Kimi)
- 自定义

**配置步骤**:
1. 选择提供商
2. 输入 API Key
3. 输入 Base URL (可选,使用默认值)
4. 输入 Model 名称
5. 点击"保存配置"

**常用配置示例**:
```
OpenAI:
- Base URL: https://api.openai.com/v1
- Model: gpt-3.5-turbo 或 gpt-4

Deepseek:
- Base URL: https://api.deepseek.com/v1
- Model: deepseek-chat

Claude:
- Base URL: https://api.anthropic.com/v1
- Model: claude-3-opus-20240229
```

### Tushare 配置
**获取 Token**:
1. 访问 https://tushare.pro
2. 注册账号
3. 获取 Token
4. 输入到应用中
5. 点击"测试连接"验证

## 🎨 高级功能

### 多 Agent 分析
**什么是多Agent分析**:
使用多个AI模型同时分析同一支股票，通过投票机制得出更可靠的分析结果。

**使用方法**:
1. 在分析页面选择"多Agent投票"模式
2. 输入股票代码
3. 点击"开始分析"
4. 查看各个Agent的分析意见
5. 查看投票结果和综合建议

### 策略回测 (Backtest) - 新增
**功能**:
- 回测历史分析信号
- 评估分析准确率
- 计算收益指标

**使用方法**:
1. 点击底部导航"持仓"旁边的按钮进入回测页面
   (或从导航图中进入 BacktestFragment)
2. 点击右下角 ▶️ 按钮
3. 配置回测参数:
   - 股票代码
   - 开始日期
   - 结束日期
4. 点击"开始回测"
5. 等待回测完成
6. 查看结果:
   - 准确率统计
   - 收益指标(总收益率、最大回撤、夏普比率)
   - 交易统计(胜率、盈亏比、交易次数)

### 股票详情页
**功能**:
- K线图查看
- 技术指标显示
- 筹码分布
- 基本面数据
- 快速分析

**技术指标**:
- MA (移动平均线)
- MACD
- RSI
- KDJ
- 布林带
- 成交量

## ⚙️ 常见问题

### Q1: 搜索功能无反应?
**A**:
1. 检查是否配置了LLM API
2. 检查网络连接
3. 检查API Key是否有效
4. 查看设置页面的连接状态

### Q2: 分析功能报错?
**A**:
1. 确认已正确配置LLM API Key
2. 确认API Key有足够的额度
3. 检查Base URL是否正确
4. 检查网络连接

### Q3: 数据加载失败?
**A**:
1. 下拉刷新重试
2. 检查网络连接
3. 如配置了Tushare,检查Token是否有效
4. 尝试使用其他数据源(在设置中切换)

### Q4: 持仓数据不准确?
**A**:
1. 手动输入交易记录时确保数据准确
2. 价格、数量、日期都要正确填写
3. 下拉刷新更新最新市值
4. 成本计算使用加权平均法

### Q5: 如何备份数据?
**A**:
- 应用数据存储在本地数据库
- 可通过Android备份功能备份
- 持仓数据可导出为CSV(待实现)

## 📊 性能优化建议

1. **定期清理历史记录**: 历史分析记录过多会影响性能
2. **合理使用分析模式**: 快速模式更省时省资源
3. **控制自选股数量**: 建议不超过50只
4. **定期刷新数据**: 避免缓存数据过期

## 🔐 隐私和安全

1. **API Key安全**:
   - API Key加密存储在本地
   - 不会上传到任何服务器

2. **数据隐私**:
   - 所有分析在本地进行
   - 持仓数据仅存储在本地
   - 不收集用户个人信息

3. **网络安全**:
   - 使用HTTPS加密通信
   - 证书验证
   - 安全的网络配置
